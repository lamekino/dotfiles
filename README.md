#### For Linux:
Requires ``stow``, ``zsh``.

To install:
```bash
$ git clone https://github.com/lamekino/dotfiles
$ cd dotfiles
$ ./install.sh
```

#### For Windows:
Requires [Powershell Core](https://github.com/powershell/powershell).

To install:
```powershell
PS> git clone "https://github.com/lamekino/dotfiles"
PS> cd .\dotfiles
PS> .\install.ps1
```
